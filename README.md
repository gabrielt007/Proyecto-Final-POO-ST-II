Este proyecto consiste en el desarrollo de una API REST utilizando Spring Boot para la gestión de un catálogo de películas. 
La aplicación permite realizar operaciones CRUD sobre la entidad Película, aplicando una arquitectura en capas, validaciones de datos y manejo adecuado de excepciones.
La API se encuentra desplegada y accesible en línea mediante la plataforma Render.

Tecnologías utilizadas

Java 17, Spring Boot, Spring Data JPA, Maven, Base de datos postgress 16, Swagger UI, HTML y JavaScript
